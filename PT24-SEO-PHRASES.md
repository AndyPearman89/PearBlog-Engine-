## OPTYMALIZACJA TREŚCI POD SEO

### On-Page Checklist:

- [x] H1 zawiera frazę (np. "Mechanik Warszawa")
- [x] Title tag zawiera frazę
- [x] Meta description zawiera frazę + CTA
- [x] URL zawiera frazę (/mechanik/warszawa/)
- [x] Pierwsze 100 słów zawiera frazę
- [x] Alt text obrazków zawiera frazę
- [x] Internal linking do innych miast
- [x] Schema.org LocalBusiness markup
- [x] NAP (Name, Address, Phone) spójne

### Content Checklist:

- [x] Min. 500 słów unique content
- [x] Naturalne użycie miasta (nie spam)
- [x] Konkretne informacje (ceny, czas, obszar)
- [x] CTA (telefon, formularz)
- [x] FAQ (3-5 pytań)
- [x] Opinie (social proof)
