# Poradnik.pro V4 HI-PRO - Quick Start Guide

Fast 5-minute setup guide for the V4 HI-PRO Content Hub homepage.

## 🚀 Quick Setup (5 minutes)

### Step 1: Create the Homepage (2 minutes)

1. **Login to WordPress Admin**
   - Navigate to your WordPress admin dashboard

2. **Create New Page**
   - Go to **Pages → Add New**
   - Enter page title: "Home" or "Poradnik Homepage"

3. **Select Template**
   - In the right sidebar, find **Page Attributes**
   - Click the **Template** dropdown
   - Select **"Poradnik.pro V4 HI-PRO Content Hub"**

4. **Publish**
   - Click **Publish** button

### Step 2: Set as Homepage (1 minute)

1. **Go to Settings**
   - Navigate to **Settings → Reading**

2. **Select Static Page**
   - Choose **"A static page"** option
   - Under **Homepage**, select your newly created page

3. **Save**
   - Click **Save Changes**

### Step 3: Verify (2 minutes)

1. **Visit Your Site**
   - Open your site's homepage in a new tab
   - You should see the new V4 HI-PRO design

2. **Check All Sections**
   - ✅ Hero with search bar
   - ✅ Quick Answer Hub
   - ✅ Category Grid
   - ✅ Featured Articles
   - ✅ Cost Hub
   - ✅ Problem Solution section
   - ✅ Internal Links
   - ✅ Trust Block
   - ✅ Final CTA
   - ✅ SEO Footer

3. **Test Responsive**
   - Resize browser window or check on mobile
   - All sections should stack nicely

## ✅ You're Done!

Your Poradnik.pro V4 HI-PRO Content Hub is now live.

## 🆘 Troubleshooting

### Template Not Showing?
- Clear WordPress cache
- Check file location: `theme/pearblog-theme/page-poradnik-v4-hipro.php`
- Refresh page editor

### Styles Look Wrong?
- Hard refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
- Check browser console for errors
- Verify CSS file exists: `assets/css/poradnik-v4-hipro.css`

### Links Not Working?
- Update URLs in template to match your site structure
- Create missing pages
- Check WordPress permalink settings
