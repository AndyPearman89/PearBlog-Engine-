## 🎓 Key Algorithms

### PT24 Lead Scoring
```
Score = Urgency(30%) + Budget(25%) + Clarity(20%) + Location(15%) + Demand(10%)

Routing:
- Score ≥80: Expert tier
- Score 50-79: Pro tier
- Score <50: Auto-reply
```

### Poradnik Content Scoring
```
Score = SEO(20%) + Engagement(20%) + CTR(20%) + Revenue(40%)

Decisions:
- Score ≥80: SCALE - Duplicate & expand
- Score 60-79: BOOST - SEO optimization
- Score 40-59: OPTIMIZE - Content improvement
- Score <40: DELETE - Consider removal
```
