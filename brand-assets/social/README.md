## 🎨 Design Guidelines

### Typography for Social
- **Headline:** Poppins Bold, 72-120px
- **Subhead:** Poppins SemiBold, 36-48px
- **Body:** Inter Regular, 24-32px
- **Minimum readable size:** 24px at intended display size

### Color Usage
- **Primary background:** Gradient #4ADE80 → #60A5FA
- **Secondary background:** Dark #0B1118
- **Text on gradient:** White #FFFFFF
- **Text on dark:** Green #4ADE80 or White
- **Accent:** Use brand secondary blue sparingly

### Logo Placement
- **Standard position:** Top-left or bottom-right
- **Size:** 60-100px height
- **Padding:** Minimum 40px from edges
- **On branded templates:** Can be larger/centered

### Safe Zones
Always keep important content within safe zones:
- **Instagram Story:** 250px from top/bottom
- **YouTube Thumbnail:** 100px from edges
- **All posts:** 5-10% margin from edges

## ✅ Quality Checklist

Before publishing social assets:
- [ ] Correct dimensions for each platform
- [ ] File size optimized (<200KB for OG, <2MB for posts)
- [ ] Text is readable at small preview sizes
- [ ] Logo/branding is visible but not overwhelming
- [ ] Colors match brand guidelines
- [ ] High contrast for accessibility
- [ ] Tested preview on actual platform
- [ ] No important content in crop zones
- [ ] Consistent style across all templates
