## 🎬 Export Guidelines

### For Designers Using:

#### Figma
1. Export at 1x, 2x, 3x for retina
2. Use PNG for raster, SVG for vector
3. Optimize exports (remove hidden layers)
4. Include style guide in Figma file

#### Adobe Illustrator
1. Artboards for each variation
2. Export as SVG (presentation attributes)
3. Export PNG at multiple resolutions
4. Save source .ai file

#### Sketch
1. Use symbol system for consistency
2. Export presets for all sizes
3. Include style library
4. Save source .sketch file

#### Photoshop
1. Smart objects for scalability
2. Export as PNG-24 (transparency)
3. Save PSD with organized layers
4. Include adjustment layers for variants

## 🔗 Integration

### WordPress Theme Integration
Location: `/theme/pearblog-theme/`

```php
// functions.php
function pearblog_get_logo($type = 'primary', $size = 'medium') {
    $logo_base = get_template_directory_uri() . '/brand-assets/logo/';

    $logos = array(
        'primary' => $logo_base . 'pearblog-logo-primary.svg',
        'dark' => $logo_base . 'pearblog-logo-dark.svg',
        'light' => $logo_base . 'pearblog-logo-light.svg',
        'icon' => $logo_base . 'pearblog-icon.svg',
        'wordmark' => $logo_base . 'pearblog-wordmark.svg',
    );

    return $logos[$type] ?? $logos['primary'];
}
```
