## 🎯 SALES TRIGGER CONDITIONS

### Automated Upgrade Triggers

**Trigger 1: High Engagement**
```php
if (
    $business->leads_received >= 3 &&
    $business->response_rate >= 70% &&
    $business->plan === 'free'
) {
    send_premium_offer($business);
}
```

**Trigger 2: Lead Conversion**
```php
if (
    $business->converted_leads >= 2 &&
    $business->plan === 'free'
) {
    send_premium_offer($business, [
        'message' => 'Widzimy że konwertujesz nasze leady. Chcesz więcej?'
    ]);
}
```

**Trigger 3: Lead Cap Hit**
```php
if (
    $business->plan === 'free' &&
    $business->leads_this_month >= $business->free_lead_cap
) {
    send_premium_offer($business, [
        'message' => 'Osiągnąłeś limit darmowych leadów. Przejdź na PREMIUM.'
    ]);
}
```

**Trigger 4: Exclusive Upsell**
```php
if (
    $business->plan === 'premium' &&
    $business->leads_received >= 10 &&
    $business->conversion_rate >= 40%
) {
    send_premium_plus_offer($business, [
        'message' => 'Konwertujesz świetnie. Chcesz exclusive leady bez konkurencji?'
    ]);
}
```

## 📈 SALES KPIs

### Track These Metrics

**Response Rate**
```
Target: > 60%

Formula: (businesses_responded / total_leads_sent) × 100

Good: 60-80%
Excellent: 80%+
```

**Conversion to PREMIUM**
```
Target: 10-20%

Formula: (premium_customers / total_active_businesses) × 100

Good: 10-15%
Excellent: 15-20%+
```

**Revenue Per Business**
```
Target: 300-600 zł/month

Average:
- FREE: 0 zł
- PREMIUM: 299 zł
- PREMIUM+: 599 zł

Blended average: ~150 zł per business
```

**Churn Rate**
```
Target: < 10% monthly

Formula: (canceled_subscriptions / total_subscriptions) × 100

Good: < 10%
Excellent: < 5%
```

**LTV (Lifetime Value)**
```
Target: > 3,600 zł (12 months retention)

Formula: avg_monthly_revenue × avg_retention_months

Good: 3,600 zł (12 months)
Excellent: 7,200 zł (24 months)
```

## 🎓 BEST PRACTICES

### Do's ✅
1. **Always show value first** (send lead, then sell)
2. **Use real numbers** (earnings, ROI, lead count)
3. **Make it easy** (one-click upgrade, fast payment)
4. **Follow up consistently** (3-7 day sequence)
5. **Track everything** (response rates, conversions, churn)

### Don'ts ❌
1. **Don't sell before value** (no cold outreach)
2. **Don't be pushy** (let the results speak)
3. **Don't forget follow-ups** (automation is key)
4. **Don't ignore churn** (retention > acquisition)
5. **Don't overcomplicate** (simple offer, clear value)
